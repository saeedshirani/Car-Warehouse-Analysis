# -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Analysis on the dataset of a car warehouse has been conducted.


#------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 In the EDA Folder, exploratory analysis results are available. 

#------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Within the Dashboard Folder, a Python file has been provided.
 Running this file allows you to access the dashboard, designed specifically for warehouse managers or stockkeepers.



In the Dashboard folder, you'll find a file named requirements.txt. 
This file contains the versions of all packages required for running the Dashboard file properly.
#------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Furthermore, there is a Dashboard performance video available for a demonstration of the dashboard's functionality.
Reports are also included in the form of REPORT_MP4 and REPORT_PDF files. 
These reports contain a simple analysis of the dataset and serve as documentation for the dashboard's performance.
